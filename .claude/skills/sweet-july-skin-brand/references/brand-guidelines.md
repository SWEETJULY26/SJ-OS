# Sweet July Skin — Brand Guidelines Reference

## Brand Story
Sweet July Skin is a skincare line from Ayesha Curry celebrating her Jamaican roots. The brand features time-tested Caribbean ingredients enhanced by modern formulas. Products turn the daily ritual of skincare into a private self-care retreat. Tagline ethos: **"Everything Irie"** — peaceful, positive, good-vibes energy.

## Typography

### Typefaces
| Font | Weight | Use |
|------|--------|-----|
| GT America Expanded | Medium | Extra headlines / large type moments / logo synonym |
| Adrianna | Demibold | Primary headlines |
| Adrianna | Regular | All body text / paragraphs |
| Saol Display | Medium | Accent use — adds drama and elegance |

### Type Styles
- **Extra Headline:** GT America Expanded Medium — ALL CAPS — 48pt / Line Height 100% / Letter Spacing 0%
- **Primary Headline:** Adrianna Demibold — Sentence Case or ALL CAPS — 38pt / Line Height 100% / Letter Spacing 0%
- **Body Text:** Adrianna Regular — 16pt / Line Height 125% / Letter Spacing 0%
- **Website only:** GT America Regular for body text on sweetjulyskin.com

### Typography Rules
- GT America is used for large type moments; synonymous with the logo
- Adrianna is the versatile workhorse — use for all headlines and body copy
- Saol Display is used sparingly for sophistication/elegance moments
- Headlines are written in UPPERCASE (sentence case acceptable as fit)

### Font Substitutions (when brand fonts are unavailable)
- GT America Expanded → Use a wide sans-serif (e.g., system wide/extended sans)
- Adrianna → Clean geometric sans-serif (e.g., Helvetica Neue, Inter)
- Saol Display → Elegant serif (e.g., Georgia, Playfair Display)

## Text Lockups

Four approved combinations for pairing type styles across print, web, and social. Each one has a specific job.

### 1. Extra + Primary Headlines (both uppercase)
Big display headline over a smaller uppercase subhead. Use for bold hero moments — campaign pages, product landing sections, cover slides.
- Display line: GT America Expanded Medium, UPPERCASE
- Secondary line: Adrianna Demibold, UPPERCASE

### 2. Extra + Primary + short caps body
Same hero pairing as above, with a short run of uppercase Adrianna Demibold standing in for body copy. Works for editorial print spreads where the "body" is really a tagline or callout, not a long paragraph.

### 3. Primary Sentence Case + Body Paragraph
For editorial content, blog posts, longer-form marketing copy.
- Heading: Adrianna Demibold, Sentence Case
- Body: Adrianna Regular, sentence case, 16pt / line height 125%

### 4. Primary Caps + Body Paragraph
For section intros, product page headers, and web layouts where the heading needs a stronger beat.
- Heading: Adrianna Demibold, UPPERCASE
- Body: Adrianna Regular, sentence case, 16pt / line height 125%

### Rules of thumb
- Never pair two uppercase blocks at the same size — hierarchy comes from size and weight, not shouting
- Extra headline (GT America Expanded) is the biggest type on the page, always
- Adrianna Demibold sentence case is softer and more editorial; UPPERCASE is louder and more directive — pick per tone
- Body copy is always Adrianna Regular (GT America Regular only on the website)

## Document & Presentation Design Principles

### Layout
- Generous white space — let the design breathe
- Bone (`#f4f0e8`) as default background
- Left-aligned or centered text — avoid right-alignment for body
- Clean, uncluttered layouts

### Slide / Deck Design
- Cover slides: Full Pava Brown or Good Youth background with white logo + text
- Content slides: Bone background, dark brown text
- Use Lignum Vitae flower as a decorative background element at 23.92°
- Section dividers: Bold color backgrounds (Pava Brown, Good Youth)

### Documents (Word / PDF)
- Bone or white background
- Pava Brown (`#8a665a`) for headings
- Good Youth (`#795d50`) for subheadings or accent text
- Black or dark brown for body text
- Thin horizontal rules in Soursop or Pava to separate sections

### Web / HTML
- Background: Bone (`#f4f0e8`)
- Primary text: Dark brown / Good Youth
- Accent / CTA buttons: Pava Brown or Good Youth
- Navigation: Inline logo, sparse and elegant

### Canva Decks
- Use warm earth palette — lead with Bone backgrounds
- Bold color accent slides using primary palette
- Typography hierarchy: GT America-equivalent for titles, clean sans for body
- Lignum Vitae flower SVG as background decoration where possible

## What to Avoid
- Do NOT use the Lignum Vitae flower without the logo nearby
- Do NOT use Saol Display as a primary body font
- Do NOT use cool, clinical, or pharmaceutical tones in copy
- Do NOT use busy, heavily patterned backgrounds
- Do NOT use neon, fluorescent, or stark colors outside the brand palette
- Do NOT use GT America Regular for body copy outside the website
