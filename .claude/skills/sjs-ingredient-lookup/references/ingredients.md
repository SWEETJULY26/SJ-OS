## Notes for Use

- **Always cite the source.** When answering questions, make clear the answer comes from the official Sweet July Skin ingredient document.
- **Don't invent ingredients.** If a product isn't in this list, say so — don't guess based on similar products.
- **Order matters.** Ingredients are listed in order of concentration (highest to lowest, with everything under 1% in any order). Mention this if relevant to the question.
- **"May Contain (+/-)" ingredients** are colorants that may or may not be present depending on the shade or batch. Flag this when answering questions about colored products.
- **Common name lookup.** Many ingredients are listed by their INCI (scientific) name. Common names worth knowing:
  - Water = Aqua
  - Aloe = Aloe Barbadensis
  - Coconut = Cocos Nucifera
  - Guava leaf = Psidium Guajava
  - Lavender = Lavandula Angustifolia
  - Lemon = Citrus Limon
  - Orange = Citrus Sinensis
  - Grapefruit = Citrus Paradisi or Citrus Grandis
  - Apricot kernel = Prunus Armeniaca
  - Sweet almond = Prunus Amygdalus Dulcis
  - Sunflower = Helianthus Annuus
  - Castor = Ricinus Communis
  - Argan = Argania Spinosa
  - Jojoba = Simmondsia Chinensis
  - Neem = Melia Azadirachta
  - Rosemary = Rosmarinus Officinalis
  - Tea tree = Melaleuca Alternifolia
  - Willow bark = Salix Nigra or Salix Alba
  - Shea butter = Butyrospermum Parkii
  - Meadowfoam = Limnanthes Alba
  - Soursop = Annona Muricata
  - Papaya = Carica Papaya
  - Banana = Musa Sapientum
  - Pineapple = Ananas Sativus
  - Wild yam = Dioscorea Villosa
  - Lychee = Litchi Chinensis
  - Lotus seed = Nelumbo Nucifera
  - Passion fruit = Passiflora Incarnata
  - Burdock = Arctium Lappa
  - Coffee = Coffea Arabica
  - Cocoa = Theobroma Cacao
  - Mango = Mangifera Indica
  - Hibiscus = Hibiscus Sabdariffa
  - Green tea = Camellia Sinensis
  - Spearmint = Mentha Spicata
  - Peppermint = Mentha Piperita
  - Turmeric = Curcuma Longa
  - Chia = Salvia Hispanica
  - Prickly pear = Opuntia Engelmannii
  - Red raspberry = Rubus Idaeus
  - Carob = Ceratonia Siliqua
  - Stevia = Stevia Rebaudiana
  - Melon = Cucumis Melo
  - Gardenia = Gardenia Florida

- **Common functional terms:**
  - Retinol / Hydroxypinacolone Retinoate = vitamin A derivatives
  - Niacinamide = vitamin B3
  - Tetrahexyldecyl Ascorbate / 3-O-Ethyl Ascorbic Acid = vitamin C derivatives
  - Tocopherol / Tocopheryl Acetate = vitamin E
  - Panthenol = pro-vitamin B5
  - Sodium Hyaluronate / Hyaluronic Acid = hydrating humectants
  - Glycolic Acid / Lactic Acid = AHAs (alpha hydroxy acids)
  - Salicylic Acid = BHA (beta hydroxy acid)
  - Ferulic Acid = antioxidant
  - Tranexamic Acid = brightening / discoloration
  - Acetyl Hexapeptide-8 / Palmitoyl Tripeptide-1 = peptides
  - Bisabolol = soothing (chamomile-derived)
  - Allantoin = soothing
  - Ferments (Lactobacillus, Leuconostoc) = probiotic / brightening / preservation
